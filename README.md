# Description

Allow to use dynamic repositories in the plugins install section.

# install

`# vendor/bin/extas init`
`# vendor/bin/extas install`

# usage

`extas.json`

```json
{
  "repositories": {
    "name": "any",
    "...": "...",
    "aliases": ["test"] 
  },
  "plugins_install": [
    {
      "repository": "test",
      "name": "some",
      "section": "tests"
    }
  ]
}
```
